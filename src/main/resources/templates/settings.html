MCP API Key

Use your API key to connect external AI assistants (like Claude) to your Car Builder data via the MCP protocol.

How to Use Your API Key

Once you have an API key, configure your MCP client with the following:

{
  "mcpServers": {
    "car-builder-vin": {
      "url": "https://your-server.com/mcp/sse",
      "headers": {
        "X-MCP-API-Key": "your-api-key-here"
      }
    }
  }
}

The MCP server provides tools to manage your vehicles, builds, parts, and more through AI assistants.
